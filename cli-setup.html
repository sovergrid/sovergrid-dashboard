Install the CLI

The SoverGrid CLI lets you deploy, manage, and monitor from any terminal or CI/CD pipeline.

# Install globally via npm
$ npm install -g @sovergrid/cli
# Authenticate
$ sovergrid login --key sgk_live_…
# Deploy your app
$ sovergrid deploy --provider akash --replicas 3

→ Provisioning 3 containers on Akash…
